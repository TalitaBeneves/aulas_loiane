O que é um Enum?

Enums é um tipo de enumeração que nós fornece uma maneira muito simples de definirmos um conjunto de constantes. Por exemplo, vamos imaginar que precisamos definir uma variável cada dia da semana. O que é um Pipe?

Para exibir corretamente os dados o Angular desenvolveu o que chamamos de pipe, os quais permitem transformar uma dados antes de ser exibido diretamente ao usuário.
No atributo data cadastro, temos uma data, que mostrada de forma bruta veremos o seguinte.

21 visitas - 2018-03-02T12:51:57

A data é exibida mas em um formato incorreto. Que tal se pudessemos melhorar esta apresentação? Usando o pipe de datas do próprio Angular, podemos deixar nosso código assim:

"{{"anuncio.visualizacoes visitas"}} - {{"anuncio.dataHoraCadastro"}}
